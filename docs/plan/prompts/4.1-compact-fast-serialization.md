# PR 4.1 — Compact & Fast Serialization

## Problem
164MB graph.json with `indent=2`, parsed via slow stdlib `json.loads()`.
Load time dominates all CLI operations (1300ms of 1500ms).
Every CLI subprocess re-parses the full JSON from scratch.

## Goal
Reduce load time from 1300ms to <200ms with zero required dependencies.

## Changes

### 1. `graphify/export.py` — Compact export
- `to_json()`: Change `json.dump(data, f, indent=2)` to `json.dump(data, f, separators=(',',':'))`
- Strip `_src`/`_tgt` from serialized edge links (redundant with source/target in node_link_data, recomputed on load in build.py)
- This alone: 164MB → ~70MB (60% reduction)

### 2. `graphify/serve.py` — Unified fast loader
- Create `_load_graph_file(graph_path: str | Path) -> nx.Graph`
- Try `orjson.loads()` if available, else `json.loads()`
- Handle .json and .json.gz (auto-detect via magic bytes)
- Support DiGraph vs Graph detection
- Expand `.json` suffix check to `.json.gz`

### 3. `graphify/__main__.py` — Use shared loader
- All 6 duplicate JSON parse blocks replaced:
  - `query`: `_raw = json.loads(gp.read_text(...))` → `_load_graph_file(graph_path)`
  - `path`: same
  - `explain`: same
  - `processes`: same
  - `cluster-only`: same
  - `update`: same
- Each currently duplicates: `json.loads(safe.read_text(...))` + `json_graph.node_link_graph(data, edges="links")`

### 4. `graphify/build.py` — Use shared loader
- `build_merge()`: Replace `json.loads()` + manual `json_graph.node_link_graph()`
- Use `_load_graph_file(graph_path)`

### 5. `graphify/benchmark.py` — Use shared loader where applicable

### 6. `tests/test_serialization.py` (NEW)
- Compact JSON round-trip: nodes, edges, hyperedges survive
- orjson available vs stdlib fallback
- .json.gz auto-detection
- DiGraph round-trip
- Edge attribute integrity after compact serialization
- `_src`/`_tgt` stripping and reconstruction

## Verification

python -m pytest tests/test_serialization.py tests/test_export.py tests/test_serve.py tests/test_build.py -q --tb=short
python -m graphify build tests/fixtures/  # verify graph.json is compact
